IPL Data Analysis

Course project submitted towards completion of the course Data Analysis with Python: Zero to Pandas by JovianML.Got to learn a lot about pandas, numpy and matplotlib, seaborn. This is the NETFLIX SHOWS Dataset (hosted on kaggle) used in this project. This dataset consists of tv shows and movies available on Netflix as of 2019. The dataset is collected from Flixable which is a third-party Netflix search engine. In 2018, they released an interesting report which shows that the number of TV shows on Netflix has nearly tripled since 2010. The streaming serviceâ€™s number of movies has decreased by more than 2,000 titles since 2010, while its number of TV shows has nearly tripled. It will be interesting to explore what all other insights can be obtained from the same dataset. Questions this analysis sought to answer Understanding what content is available in different countries Identifying similar content by matching text-based features Network analysis of Actors / Directors and find interesting insights Is Netflix has increasingly focusing on TV rather than movies in recent years.
